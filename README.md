# 🎂 Bake a Wish

An interactive and customizable **Birthday Cake Web App** built with **HTML, CSS, and JavaScript**. Personalize your cake, light and blow out candles, launch confetti and fireworks, play a birthday tune, and download your celebration as a PNG—all directly in the browser with no installation required.

---
Live Preview 👉 : https://godeshwari.github.io/Cake_Animation/

---
## 🌟 Features

- 🎉 Personalized Birthday Name Banner
- 🎂 5 Cake Flavors
  - Vanilla
  - Chocolate
  - Strawberry
  - Mint
  - Lemon
- 🕯️ Adjustable Candles (1–16)
- 🔥 Click individual candles to blow them out
- 💨 "Make a Wish" button to blow out all candles
- 🕯️ Relight all candles instantly
- 🌈 4 Beautiful Background Themes
  - Candy Sky
  - Starlit Night
  - Garden Party
  - Disco Confetti
- 🌙 Dark & Light Mode
- 🎊 Confetti Celebration
- 🎆 Fireworks Animation
- 🎈 Floating Balloons
- 🎵 Built-in Birthday Tune using the Web Audio API
- 📸 Download Celebration as PNG
- 💾 Automatic Settings Saving with Local Storage
- 📱 Fully Responsive Design
- 🎛️ Mobile-Friendly Collapsible Control Panel
